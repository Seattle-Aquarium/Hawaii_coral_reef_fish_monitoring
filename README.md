# Seattle_Aquarium_Hawaii_coral_reef_fish_monitoring
This repository contains all code and data associated with "_A unique 100 Meter Underwater Survey Method Documents Changes in Abundance, Richness, and Community Structure of Hawai πi Reef Fishes_", published in Frontiers in Marine Science in 2022. A pdf of the publication can be viewed and downloaded here: https://www.frontiersin.org/articles/10.3389/fmars.2022.892261/full
